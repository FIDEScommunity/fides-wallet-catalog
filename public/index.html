FIDES Wallet Catalog API

Read-only API over data/aggregated.json.

  • GET /api/public/wallet — list wallets (pagination + filters)
  • GET /api/public/wallet/{orgId}/{walletId} — one wallet (encode org: in path, e.g. org%3Aanimo)
  • GET /api/public/providers
  • GET /api/public/stats
  • GET /api/public/filter-options
  • GET /api/public/api-docs — OpenAPI 3.1 (JSON)
  • Swagger UI
